# 后端开发计划 - MOD-01 指纹引擎

## 文档信息

| 字段 | 内容 |
|------|------|
| **模块编号** | MOD-01 |
| **模块名称** | 指纹引擎 |
| **对应架构** | docs/v1/02-architecture/03-mod-01-fingerprint.md |
| **优先级** | P0 |

---

## 1. 模块概述

- 负责生成确定性浏览器指纹
- 负责随机指纹生成与一致性校验
- 为实例管理与浏览器运行时提供输入基线

## 2. 接口清单

| 接口编号 | 方法 |
|---------|------|
| API-M1-001 | Generate(seed, country) |
| API-M1-002 | GenerateRandom(country) |
| API-M1-003 | Validate(fingerprint) |

## 3. 开发任务拆分

| 任务编号 | 内容 | 对应接口 |
|---------|------|---------|
| T-01 | 定义 `Fingerprint` / `CountryConfig` 数据结构 | API-M1-001~003 |
| T-02 | 建立国家配置表与默认配置加载 | API-M1-001~002 |
| T-03 | 实现基于 seed 的确定性随机数生成 | API-M1-001 |
| T-04 | 实现 `Generate` 主流程 | API-M1-001 |
| T-05 | 实现 `GenerateRandom` | API-M1-002 |
| T-06 | 实现时区/语言/平台/GPU 校验规则 | API-M1-003 |
| T-07 | 补充生成器单元测试 | API-M1-001~002 |
| T-08 | 补充校验器单元测试 | API-M1-003 |

## 4. 验收清单

- `seed + country` 生成结果稳定可复现
- 非法国家、非法 seed、非法组合均能显式报错
- 单元测试覆盖生成与校验主路径
