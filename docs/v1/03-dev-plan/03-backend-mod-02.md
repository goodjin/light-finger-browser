# 后端开发计划 - MOD-02 实例管理

## 文档信息

| 字段 | 内容 |
|------|------|
| **模块编号** | MOD-02 |
| **模块名称** | 实例管理 |
| **对应架构** | docs/v1/02-architecture/03-mod-02-instance.md |
| **优先级** | P0 |

---

## 1. 模块概述

- 管理浏览器实例生命周期、端口、CDP、存储
- 承接指纹、代理、浏览器运行时三类依赖

## 2. 接口清单

| 接口编号 | 方法 |
|---------|------|
| API-M2-001 | Create(cfg) |
| API-M2-002 | Destroy(id) |
| API-M2-003 | Get(id) |
| API-M2-004 | List(filter) |
| API-M2-005 | GetCDPClient(id) |

## 3. 开发任务拆分

| 任务编号 | 内容 | 对应接口 |
|---------|------|---------|
| T-01 | 定义 `BrowserInstance` / `InstanceConfig` / 状态枚举 | API-M2-001~005 |
| T-02 | 实现实例存储接口与基础过滤 | API-M2-003~004 |
| T-03 | 实现端口分配器 | API-M2-001 |
| T-04 | 实现浏览器进程启动流程 | API-M2-001 |
| T-05 | 实现进程停止与清理流程 | API-M2-002 |
| T-06 | 实现 `Create` / `Destroy` 编排 | API-M2-001~002 |
| T-07 | 实现 `Get` / `List` 查询接口 | API-M2-003~004 |
| T-08 | 实现 CDP 连接与缓存 | API-M2-005 |
| T-09 | 增加状态同步与异常退出监测 | API-M2-001~005 |
| T-10 | 补充管理器、进程与 CDP 单元测试 | API-M2-001~005 |

## 4. 验收清单

- 实例可创建、可销毁、可重连 CDP
- 异常退出后状态不会长期停留在 `running`
- 端口、用户目录与实例 ID 一一对应
